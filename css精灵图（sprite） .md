# css精灵图（sprite）

### 精灵图作用：为了减少服务器的接受和发送请求的次数，提高页面的加载速度

**核心：**

1. **主要针对背景图片使用**
2.  **主要借助背景位置background-position移动距离就是x，y坐标**
3. **一般情况下精灵图都是负值**

### 字体图标的优点：

1. 轻量级
2. 灵活性
3. 兼容性
- **结构与样式简单的小图标用字体图标**
- **结构与样式复杂的用精灵图**
